# Descrição: Sistema bancário      #

Aplicação simples desenvolvida simulando um sistema de banco com menu e opções de:

        1. Criar conta
        2. Criar usuário
        3. Acessar conta
        4. Depositar
        5. Sacar
        6. Consultar saldo
        7. Exibir contas cadastradas
        8. Sair

Criado com base em um desafio em bootcamp.
